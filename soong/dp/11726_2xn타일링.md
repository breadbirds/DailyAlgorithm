## 제출 코드

- `1, 2, 3, 5, 8 ...` 의 형식으로 진행되므로, 
    `dp[4] = dp[2] + dp[3]`
- 점화식 : `dp[i] = dp[i - 1] + dp[i - 2]`

```java
int[] dp = new int[n + 2];
dp[1] = 1;
dp[2] = 2;

for (int i = 3; i <= n; i++) {
	dp[i] = dp[i - 1] + dp[i - 2];
}

System.out.println(dp[n]);
```
