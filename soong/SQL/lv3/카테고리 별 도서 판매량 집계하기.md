# 카테고리 별 도서 판매량 집계하기

## 제출 코드

이번엔 너무 어렵게 생각해서 틀린 코드
어렵다! ㅎㅣㅇ

```mysql
SELECT B.CATEGORY, SUM(S.SALES) as TOTAL_SALES
from BOOK B
JOIN BOOK_SALES S
ON B.BOOK_ID = S.BOOK_ID
WHERE SALES_DATE BETWEEN '2022-01-01' AND '2022-01-31'
GROUP BY B.CATEGORY
ORDER BY B.CATEGORY
```
