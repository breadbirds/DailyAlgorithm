### 목표 : 매일 자유롭게 알고리즘 1개 이상 풀고 커밋해서 잔디 심기

- 당일 자정 넘으면 자동으로 리드미 변경됩니당
- git config 필요합니다 (깃헙 아이디, 메일이 등록돼있어야 깃헙 액션 동작)
```
git config user.name "깃헙 아이디"
git config user.email "깃헙 이메일"
```
